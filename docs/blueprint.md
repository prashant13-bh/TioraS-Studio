# **App Name**: TioraS AI Menswear

## Core Features:

- AI Design Generation: Allow users to generate custom clothing designs using Google Gemini Pro, acting as a tool for visual creativity and personalization of apparel.
- Product Catalog & Filtering: Display a real-time product catalog with dynamic filtering and sorting options to enhance product discovery.
- Shopping Cart and Checkout: Enable users to add products to a shopping cart and complete the checkout process with shipping options.
- Order Management: Implement order management capabilities including order history, and guest tracking.
- Admin Dashboard with Analytics: Provide an admin dashboard for monitoring analytics and managing orders.
- Design Review System: Allows an admin to review designs and approve them for sale to comply with the T&Cs.
- Database integration: Integration with SQLite for product data, user designs, and order history, and the use of Prisma ORM.

## Style Guidelines:

- Primary color: Saturated mustard yellow (#F59E0B) to reflect the brand color.
- Background color: Desaturated light mustard (#FAF7F0) for a soft, inviting backdrop.
- Accent color: Light amber (#D97706) to provide visual contrast and direct user attention to key interactive elements.
- Headline font: 'Space Grotesk', sans-serif. Space Grotesk will give the design a contemporary edge.
- Body font: 'Inter', sans-serif. Inter will complement the more stylistic headline font while facilitating comfortable reading.
- Use minimalist icons from Lucide React for clear, modern UI elements.
- Apply subtle animations using Framer Motion for enhanced user experience, such as hover effects and transitions.
